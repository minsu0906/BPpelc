# 채용 가산점 레이더

Gemini API 키 하나로 대기업·공기업 채용공고를 웹 검색 기반으로 살펴보고, 준비할 자격증과 가산점 포인트를 정리해주는 정적 웹앱입니다.

기본값은 `최신 공고 1건 기준 분석`이고, 필요할 때는 `특정 기간 공고 기준 분석`으로 전환할 수 있습니다.

## 포함된 기능

- 관심 직무, 대기업/공기업 범위 설정
- 최신 공고 1건 기준 분석
- 특정 시작일/종료일 기준 기간 분석
- Gemini `google_search` 도구를 사용한 웹 검색 기반 분석
- 자격증 추천, 가산점 포인트, 최근 공고 예시, 4주 준비 플랜 출력
- 검색 근거 링크 목록 표시
- 직무별 기본 준비 로드맵 제공
- API 키가 없어도 화면 구조를 볼 수 있는 데모 결과 제공

## 실행 방법

이 프로젝트는 빌드 없이 정적 파일만으로 동작합니다.

1. PowerShell에서 프로젝트 폴더로 이동합니다.
2. 아래 명령으로 간단한 로컬 서버를 실행합니다.

```powershell
python -m http.server 4173
```

3. 브라우저에서 `http://127.0.0.1:4173` 를 엽니다.

## 사용 방법

1. Google AI Studio에서 발급한 Gemini API 키를 입력합니다.
2. 관심 직무와 분석 대상을 선택합니다.
3. 기본은 `최신 공고 1건` 기준으로 분석합니다.
4. 특정 시기의 공고 흐름을 보고 싶다면 `특정 기간 공고`를 선택하고 날짜를 입력합니다.
5. 필요하면 관심 기업·기관을 쉼표로 적습니다.
6. `공고 기준 분석하기`를 누르면 결과와 근거 링크가 표시됩니다.

## 참고

- 이 앱은 브라우저에서 직접 Gemini API를 호출합니다. 프로토타입 용도에는 적합하지만, 실제 서비스 배포 시에는 서버 프록시나 서버리스 함수로 API 키를 숨기는 구성이 더 안전합니다.
- 가산점 정보는 기관별 채용 공고문 원문이 최종 기준입니다.
- 현재 모델은 `gemini-3-flash-preview`로 설정되어 있습니다. 사용 가능 모델이 바뀌면 `app.js`의 `MODEL_NAME`을 수정하면 됩니다.
