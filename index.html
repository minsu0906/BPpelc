Job Notice Summary

공고 기준으로 바로 정리

원하는 기업 1곳만 입력하면 됩니다. 최신 공고 1회는 Gemini로 찾고, 특정 공고는 PDF를 브라우저에서 직접 읽어 정리합니다.

입력

필수 정보

공고 기준
학력 구분

결과

공고 분석 결과

대기 중

기업과 조건을 입력하면 공고 기준으로 정리한 결과가 아래에 표시됩니다.

기준 공고가 여기에 표시됩니다

분석을 실행하면 어떤 공고를 기준으로 정리했는지 먼저 보여드립니다.

01

정보

지원 자격, 고용 형태, 모집 인원 같은 기본 정보가 정리됩니다.

02

채용 절차

전형 단계별 일정, 평가 항목, 배수, 배점, 동점자 처리 기준을 정리합니다.

03

주요 가점 항목

공고에 실제로 적힌 가점과 우대 사항을 중심으로 보여드립니다.

근거

출처

최신 공고 모드는 검색 링크를, 업로드 공고 모드는 업로드한 파일 기준임을 표시합니다.
