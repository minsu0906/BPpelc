핵심만 입력하고
공고 기준으로 바로 정리

원하는 기업 1곳과 공고 기준만 선택하면, 결과를 `정보`, `채용 절차`, `주요 가점 항목` 세 가지로 나눠 읽기 쉽게 정리합니다.

입력

핵심 정보

공고 기준
학력 구분
대상 범위
세부 옵션

API 키는 브라우저 로컬 저장소에만 저장됩니다.

결과

공고 분석 결과

대기 중

기업과 조건을 입력하면 공고 기준으로 정리된 결과가 아래에 표시됩니다.

기준 공고가 여기에 표시됩니다

분석을 실행하면 기준이 된 공고명과 핵심 메모를 먼저 보여드립니다.

01

정보

지원 자격, 고용 형태, 모집 인원 같은 핵심 정보가 정리됩니다.

02

채용 절차

전형 단계별 평가 항목, 배수, 배점, 동점자 처리 기준, 일정이 정리됩니다.

03

주요 가점 항목

가점이나 우대 사항이 명시된 항목만 우선적으로 보여드립니다.

출처
